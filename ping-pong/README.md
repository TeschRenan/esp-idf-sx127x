# ping-pong Example   
Send data from primary to secondary.   
In the secondary, the characters are converted and sent back.

# Response time   
- 31  Byte:  90 MillSec   
- 63  Byte: 120 MillSec   
- 127 Byte: 250 MillSec   
- 255 Byte: 450 MillSec
